perls2: Perception and Robot Learning System
============================================

Index for documentation related to perls2. perls2 is a library that connects simulators, robots, cameras, sensors and controllers for robot learning experiments. It allows for easily switching between simulated and real robot experiments, and provides access to documented and validated robot controllers in joint space and operational space.
